- Wrapper/package code in this repository is licensed under MIT.
- Bundled extension binaries keep their upstream license/terms.
- This file tracks where each bundled binary comes from and where to review its upstream terms.

## Maintainer Steps

When adding/updating extension packages:

1. Update this file with an entry for the extension alias.
2. For non-default third-party terms, update the extension README `License` section to distinguish wrapper/package license from bundled binary terms.
3. Run `python scripts/maintainer.py verify-licensing` before opening the PR.
